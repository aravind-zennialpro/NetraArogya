# NetraArogya 💊✨

NetraArogya is a polished, production-grade web application designed to help users identify and understand their medicines. By leveraging the advanced vision and linguistic capabilities of **Google Gemini AI**, the app provides instant, detailed information about medicine tablet strips from a simple photo or a text search.

## 🚀 Key Features

*   **Multimodal AI Analysis**: Upload or capture a photo of a medicine strip (front or back) to extract brand names, chemical compositions, and medical details using high-precision OCR and Vision AI.
*   **Search by Text**: Quickly find information about any medicine by typing its name or active ingredients.
*   **Multilingual Interface**: View medicine information in 10 major languages, including English, Hindi (हिन्दी), Marathi (मराठी), Bengali (বাংলা), and more.
*   **Comprehensive Information**:
    *   **Main Purpose**: What the medicine is primarily used for.
    *   **How it Works**: Simple explanation of the mechanism of action.
    *   **Usage Guidance**: General dosage instructions and timing.
    *   **Safety First**: Critical warnings, precautions, and potential side effects.
    *   **Simple Understanding**: A one-sentence AI summary for non-technical users.
*   **Scan History**: Automatically records and stores your scans and searches to a MongoDB database for easy retrieval later.
*   **Premium UI/UX**: Feature a modern, dark/light theme inspired by Canva AI, complete with animated logos and fluid transitions.

## 🛠️ Technology Stack

*   **Frontend**: React 19, TypeScript, Tailwind CSS, Framer Motion, Lucide React icons.
*   **Backend**: Node.js, Express.js.
*   **AI Engine**: Google Gemini 3.1 Flash Lite API.
*   **Database**: MongoDB (via Mongoose).
*   **Tooling**: Vite, TSX, ESLint.

## ⚙️ Configuration

The application requires the following environment variables defined in a `.env` file (or platform settings):

```env
GEMINI_API_KEY=your_google_gemini_api_key
MONGODB_URI=your_mongodb_connection_string
```

## 📦 Installation & Deployment

1.  **Clone the project**.
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Setup Environment**: Add your API keys to `.env`.
4.  **Run Development Server**:
    ```bash
    npm run dev
    ```
5.  **Build for Production**:
    ```bash
    npm run build
    ```
